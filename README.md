# 项目介绍

- 该项目使用了 storybook 搭建官网
- 技术栈：React Hooks + TypeScript

# storybook install

安装

```$xslt
npx -p @storybook/cli sb init --type react_scripts
```

运行

```$xslt
npm run storybook
```

# 选择的一些库

- 动画：react-transition-group
- 图标：@fortawesome/react-fontawesome
- 对 className 操作：classnames
